\begin{onecolentry}
  Tech: Vue.js, Feathers.js, Feathers-Vuex, MongoDB, Mongoose, Node.js, Cloudflare, Vultr VPS

  \begin{highlights}
  \item Designed and delivered a full-stack web application as sole developer for \href{https://provq.com}{ProVQ}, an automotive apprenticeship training provider, to digitise their paper-based applicant screening process

  \item Built a test engine supporting multiple choice, text input and numeric answer types, with configurable pass criteria and a single retake per test
  \item Created a staff portal with applicant search, test management, applicant state reset, and CSV data export
  \item Integrated with the PICS Learner Management System (LMS) to automatically enrol successful applicants
  \item Configured and deployed to a Vultr Virtual Private Server (VPS) with Cloudflare for DNS and DDoS protection
  \end{highlights}
